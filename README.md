# Name

めっけた      シニア層向けのYoutube動画検索サイト



# Description

Youtube  data APIを用い、動画検索機能を実装しております。シニア層向けということでシンプルなUIを意識し、機能を詰め込みすぎないよう留意しました。





# Dependency

ruby 2.5.5

rails 5.2.3

jquery 

youtube data API
